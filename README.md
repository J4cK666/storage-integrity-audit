# storage-integrity-audit
Keyword-based storage integrity audit system for fast data retrieval, integrity verification and security auditing in cloud environments.



## 系统框架
前端使用html+css+js创建界面
后端使用fastAPI框架（Python）
SQLite轻量级数据库
Cloud模拟云端操作


## 审计算法：
使用双线性配对（charm_crypto）
环境配置：依赖Linux系统，PBC1.0.0，GMP等，在Windows系统无法运行。
https://jhuisi.github.io/charm/install_source.html
